TEOS AI Helper

This is an experimental addon, that relies on Odoo AI server to answer questions about the Odoo database.

Informations
  • The AI prompt and models included in the query sent can be modified in the general settings:
  • Using the AI models chosen, a table definition is sent with the prompt to help the AI better understand our database structure.
  • This is just an experimental addon. The AI might hallucinate in some cases and give wrong answers.
  • Please contact us with any ideas or feedback.
